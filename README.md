# MNNPC

Objects and functions for classifying field data according to
Minnesota’s (MN) Native Plant Community (NPC) Classification System
version 2.0.

## Overview

This package is integrated into the **R** **M**odular **A**nalysis of
**V**egetation **I**nformation **S**ystem (`RMAVIS`) R Shiny web
application. To classify field data, please follow the instructions on
the [RMAVIS app](https://connect-apps.ceh.ac.uk/RMAVIS/).

For more information on the MN NPC Classification System, please visit
the [Minnesota Department of Natural Resource’s
website](https://www.dnr.state.mn.us/npc/index.html).

Taxonomy for this package is based on
[MNTaxa](https://www.dnr.state.mn.us/eco/mbs/plant-lists.html): The
State of Minnesota Vascular Plant Checklist.

## Installation

You can install the package from GitHub with:

``` r
# install.packages("devtools")
devtools::install_github("MN-DNR-MBS/MNNPC")
```
